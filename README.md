# TIMER BD ⏱️

TIMER BD is a modern countdown and deadline management web application that helps users track important events efficiently. Users can add custom events, view real-time countdowns, and manage all deadlines in one clean, responsive interface.

## Features

- Create and manage multiple countdown events
- Real-time countdown updates
- Events persist using browser localStorage
- Manual event creation via modal form
- Delete individual events or clear all at once
- Responsive grid layout for all screen sizes
- Clean, modern UI with smooth animations
- Bengali language support in key UI areas

## Tech Stack

- React with TypeScript
- Tailwind CSS
- Font Awesome icons
- LocalStorage for data persistence
